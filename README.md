A Python script that scrapes the list of Cantonal Universities in Switzerland from Wikipedia and saves them into a clean CSV file.

##  Technologies
* **Python**
* **BeautifulSoup4** (for extracting data from HTML)
* **Pandas** (for creating the CSV file)
* **Requests** (for connecting to the website)

##  How to run it

1.  Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```

2.  Run the script:
    ```bash
    python scraper.py
    ```

3.  Done! You will see a new file called `swiss_universities.csv` with the data.
